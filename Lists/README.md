# Swift Wombat examples

Welcome. You are browsing an example Xcode project/playground created for this [Swift Wombat tutorial](https://blog.kamil.id/swiftwombat/how-to-display-a-scrollable-list-in-swiftui-using-the-list-view/).

## How to download this example project?

You can use one of two ways:

- Clone the whole repository using terminal

```bash
git clone https://github.com/kamilpowalowski/swiftwombat-projects.git
```

or

- Download the directory using this [DownGit URL](https://downgit.github.io/#/home?url=https://github.com/kamilpowalowski/swiftwombat-projects/tree/main/Lists).
